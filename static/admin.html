Wake Word Diagnostics
Stage 1 — Coral Edge TPU (~1-3ms) — Runs a dedicated keyword classifier on the TPU. Requires edgetpu_compiler.
Stage 1b — CPU TFLite (~3-8ms) — Same classifier on CPU. Used when Edge TPU model is not available.
Stage 2 — Verifier (~2ms) — Cosine-similarity check against trained centroid. Fast pre-filter.
Stage 3 — Silero VAD (~14ms) — Filters out silence. Only speech-containing audio reaches Whisper.
Stage 4 — Whisper STT (~150ms) — Full transcription fallback. Only fires when VAD detects speech.

How to train: Click "Train Wake Word Model" above. Nova generates synthetic audio via Piper TTS, trains a cosine-similarity verifier and a TFLite classifier, and optionally compiles for Edge TPU if the compiler is installed. No microphone needed — fully automatic.
